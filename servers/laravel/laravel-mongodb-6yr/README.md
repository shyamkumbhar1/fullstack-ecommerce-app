# Laravel + MongoDB Server (2018)

This project demonstrates senior-level experience with legacy Laravel and MongoDB from 2018.

## Technology Stack

- **PHP:** v7.1+
- **Laravel:** v5.8
- **MongoDB:** via jenssegers/mongodb v3.6
- **Composer:** Standard

## Features

- Laravel v5.8 patterns (legacy)
- Eloquent ORM with MongoDB
- Service providers
- Traditional middleware
- JWT authentication
- Legacy API patterns

## Installation

```bash
composer install
```

## Configuration

Create `.env` file:

```env
APP_ENV=local
APP_DEBUG=true
DB_CONNECTION=mongodb
MONGODB_URI=mongodb://localhost:27017
MONGODB_DATABASE=ecommerce
JWT_SECRET=your_jwt_secret_key_here
```

## Running

```bash
php artisan serve --port=5013
```

Server runs on port **5013**
