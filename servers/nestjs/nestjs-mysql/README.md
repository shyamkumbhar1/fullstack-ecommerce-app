# NestJS + MySQL Server (2024)

This project demonstrates modern experience with NestJS and MySQL from 2024.

## Technology Stack

- **Node.js:** v18+
- **NestJS:** v10+
- **MySQL:** via TypeORM v0.3.17
- **TypeScript:** v5+

## Features

- Modern NestJS patterns
- TypeORM integration
- TypeScript throughout
- Decorator-based architecture
- Dependency injection
- JWT authentication

## Installation

```bash
npm install
```

## Configuration

Create `.env` file:

```env
NODE_ENV=development
PORT=5003
DB_HOST=localhost
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=
DB_DATABASE=ecommerce
JWT_SECRET=your_jwt_secret_key_here
```

## Running

```bash
npm run start:dev
```

Server runs on port **5003**
