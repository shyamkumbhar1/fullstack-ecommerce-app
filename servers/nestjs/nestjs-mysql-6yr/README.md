# NestJS + MySQL Server (2018)

This project demonstrates senior-level experience with legacy NestJS and MySQL from 2018.

## Technology Stack

- **Node.js:** v10+
- **NestJS:** v6
- **MySQL:** via TypeORM v0.2.22
- **TypeScript:** v3.7

## Features

- NestJS v6 patterns (early framework)
- TypeORM v0.2 (early version)
- TypeScript v3.7
- Basic decorator patterns
- JWT authentication

## Installation

```bash
npm install
```

## Configuration

Create `.env` file:

```env
NODE_ENV=development
PORT=5011
DB_HOST=localhost
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=
DB_DATABASE=ecommerce
JWT_SECRET=your_jwt_secret_key_here
```

## Running

```bash
npm run start:dev
```

Server runs on port **5011**
