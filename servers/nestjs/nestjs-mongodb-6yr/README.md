# NestJS + MongoDB Server (2018)

This project demonstrates senior-level experience with legacy NestJS and MongoDB from 2018.

## Technology Stack

- **Node.js:** v10+
- **NestJS:** v6
- **MongoDB:** v3+ (Mongoose v5)
- **TypeScript:** v3.7

## Features

- NestJS v6 patterns (early framework)
- TypeScript v3.7
- Basic decorator patterns
- Early dependency injection
- Traditional error handling
- JWT authentication

## Installation

```bash
npm install
```

## Configuration

Create `.env` file:

```env
NODE_ENV=development
PORT=5009
MONGO_URI=mongodb://localhost:27017/ecommerce
JWT_SECRET=your_jwt_secret_key_here
```

## Running

```bash
npm run start:dev
```

Server runs on port **5009**
