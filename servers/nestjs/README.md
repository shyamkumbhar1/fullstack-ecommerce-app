# NestJS + MongoDB Server (2024)

This project demonstrates modern experience with NestJS and MongoDB from 2024.

## Technology Stack

- **Node.js:** v18+
- **NestJS:** v10+
- **MongoDB:** v7+ (Mongoose v7+)
- **TypeScript:** v5+

## Features

- Modern NestJS patterns
- TypeScript throughout
- Decorator-based architecture
- Dependency injection
- Modern error handling
- JWT authentication with Passport

## Installation

```bash
npm install
```

```bash
npm run start:dev
```

Server runs on port **5002**
