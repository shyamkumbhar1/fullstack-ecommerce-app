# NestJS + MongoDB Server (2021)

This project demonstrates mid-level experience with NestJS and MongoDB from 2021.

## Technology Stack

- **Node.js:** v14+
- **NestJS:** v8
- **MongoDB:** v4+ (Mongoose v6)
- **TypeScript:** v4.5

## Features

- NestJS v8 patterns
- TypeScript v4.5
- Decorator-based architecture
- Dependency injection
- Basic error handling
- JWT authentication

## Installation

```bash
npm install
```

## Configuration

Create `.env` file:

```env
NODE_ENV=development
PORT=5008
MONGO_URI=mongodb://localhost:27017/ecommerce
JWT_SECRET=your_jwt_secret_key_here
```

## Running

```bash
npm run start:dev
```

Server runs on port **5008**
