# NestJS + MySQL Server (2021)

This project demonstrates mid-level experience with NestJS and MySQL from 2021.

## Technology Stack

- **Node.js:** v14+
- **NestJS:** v8
- **MySQL:** via TypeORM v0.2.41
- **TypeScript:** v4.5

## Features

- NestJS v8 patterns
- TypeORM v0.2
- TypeScript v4.5
- Decorator-based architecture
- JWT authentication

## Installation

```bash
npm install
```

## Configuration

Create `.env` file:

```env
NODE_ENV=development
PORT=5010
DB_HOST=localhost
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=
DB_DATABASE=ecommerce
JWT_SECRET=your_jwt_secret_key_here
```

## Running

```bash
npm run start:dev
```

Server runs on port **5010**
